## 各パターンの手順

### 1. NG波形を解析したい

1. 解析したい `.dat` ファイルを `波形解析/projects/<製品名>/data/` に置く
2. `NG解析.bat` を実行する（またはAIエージェントに依頼する）
3. 結果が `波形解析/projects/<製品名>/output/` に出力される

→ 詳細: [波形解析/README.md](波形解析/README.md)

### 2. datをCSVに変換したい

1. 変換したい `.dat` ファイルのパスを確認する
2. `波形解析/scripts/dat_to_csv.py` を実行する
3. 同じフォルダにCSVファイルが生成される

→ 詳細: [波形解析/README.md](波形解析/README.md)

### 3. 波形ビューアで見たい

1. CSVファイルを用意する（なければdatから変換する）
2. `波形解析/scripts/波形ビューア起動.bat` を実行する
3. 見たい信号を選んでグラフ表示する

→ 詳細: [波形解析/README.md](波形解析/README.md)

### 4. オシロの画面をキャプチャしたい

1. オシロスコープとPCをUSBケーブルで接続する
2. `波形取得/画面取得.bat` を実行する
3. `波形取得/画像ファイル/` にPNGが保存される

→ 詳細: [波形取得/README.md](波形取得/README.md)

### 5. uf.c/uf.hを編集したい

1. 編集前に必ずバックアップを取る（`一時/` フォルダに保存）
2. uf.hは構造体内部の変数定義のみ編集する
3. uf.hを編集したらmain.hにもコピーする

→ 詳細: [UF作成/README.md](UF作成/README.md)

### 6. 画像をAIに見せたい

1. MCPサーバーが起動していることを確認する（VS Code再起動で有効化）
2. `list_images(フォルダパス)` で画像一覧を取得する
3. `view_image(ファイルパス)` で画像をAIに渡す

→ 詳細: [mcp/README.md](mcp/README.md)

### 7. AIエージェント（IMF）に作業を依頼したい

このプロジェクトには4人のAIエージェント（IMFチーム）がいる。作業内容に応じて適切なメンバーに依頼する。

#### メンバーと使い分け

| メンバー | 得意なこと | こういう時に頼む |
|---------|-----------|----------------|
| **フィリップス** (Kingpin) | 指揮・方針決定 | 作業の方針を決めてほしい、複数の作業をまとめて進めたい |
| **コール** (Shadow) | 実装・解析・データ処理 | コードを書いてほしい、波形を解析してほしい、データを処理してほしい |
| **バーンズ** (Doc) | 技術検証・レビュー | コードが正しいか確認してほしい、データの裏取りをしてほしい |
| **ヴォルコワ** (Siren) | 報告書チェック | 報告書の最終確認をしてほしい |

#### 依頼の流れ

1. **フィリップスに指示を出す** — まずチームリーダーに作業内容を伝える
2. **フィリップスが適切なメンバーに振る** — 実装ならコール、検証ならバーンズへ
3. **結果が戻ってくる** — コール→バーンズ→フィリップス→ヴォルコワの順でチェックされる

> **ヒント**: 簡単な作業（ファイル変換、単純な解析）はコールに直接依頼してもOK。

## 新しいプロジェクトの追加方法

1. `波形解析/projects/<製品名>/` フォルダを作成する
2. `config.yaml` を作成する（`projects/919D/config.yaml` を雛形にする）
3. NG番号辞書、カラム名、テストマーカー等を製品に合わせて設定する
4. `data/`、`output/`、`docs/` フォルダを用意する

## 前提条件

| 項目 | 必要なもの |
|------|-----------|
| Python | 3.12以上 |
| 波形解析 | `pip install pandas pyyaml matplotlib` |
| 波形ビューア | `pip install PySide6 pyqtgraph` |
| 波形取得 | `pip install pyvisa pyvisa-py` + USBケーブル |
| MCP画像サーバー | `pip install fastmcp Pillow` |
