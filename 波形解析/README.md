# 波形解析

テストデータ（CANoeログ、オシロスコープ波形等）の解析ツール・プロジェクト管理。

## フォルダ構成

```
波形解析/
├── scripts/              # 解析スクリプト（プロジェクトに応じて追加）
├── projects/             # プロジェクトごとのデータ・結果
│   └── <プロジェクト名>/
│       └── output/       # 解析結果の出力先
└── README.md
```

## 想定ツール

- CANoeログ（.blf / .asc）の解析スクリプト
- テスト結果の可視化ツール
- NG判定の自動化スクリプト
- 波形ビューア

## 使い方

プロジェクトごとに `projects/<プロジェクト名>/` フォルダを作り、データと結果を管理する。
汎用スクリプトは `scripts/` に配置する。
