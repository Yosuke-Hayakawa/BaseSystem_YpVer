# ツール

プロジェクト横断で使う便利ツール。

## ツール一覧

| ファイル | 説明 |
|----------|------|
| `restore_chat_sessions.py` | VS Code Copilot Chatの会話セッションを旧ワークスペースから新ワークスペースに復旧する |
| `restore_chat_sessions.bat` | 上記スクリプトのバッチラッパー（ダブルクリックで実行） |

## restore_chat_sessions.py

ワークスペースを移行した際に、旧ワークスペースのCopilot Chatの会話履歴を新ワークスペースにマージするスクリプト。

### 使い方

1. **VS Codeを完全に閉じる**（起動中は実行不可）
2. `restore_chat_sessions.bat` をダブルクリックする
3. 完了メッセージを確認してVS Codeを再起動する

### 注意事項

- 実行前にVS Codeが完全に終了していることを確認すること
- スクリプト内のパス定数は環境に合わせて変更が必要
